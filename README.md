🏥 Hospital Registration Form
📌 Overview
This project is a simple and responsive Hospital Registration Form built using HTML and CSS.
It allows patients to enter their personal details, select a department, and describe symptoms before registration.
✨ Features
🧑 Patient information input (Name, DOB, Gender)
📞 Contact details (Phone & Email)
🏠 Address field
🏥 Department selection
📝 Symptoms description
🎨 Clean and modern UI design
📱 Responsive layout
🛠️ Technologies Used
HTML5 – Structure of the form
CSS3 – Styling and layout
📂 Project Structure

project-folder/
│── index.html   # Main file containing form and styling
⚙️ How It Works
User fills in personal details
Selects gender and department
Enters symptoms
Clicks Register button
Currently, the form does not store or process data (no backend integration).
▶️ How to Run
Copy the code into a file named index.html
Open it in any web browser
Fill the form and click Register
🎨 UI Highlights
Gradient background for better appearance
Centered card-style form
Rounded input fields
Smooth and simple design
⚠️ Limitations
No data storage
No validation beyond basic HTML
No backend connectivity
No confirmation message
🔮 Future Improvements
Add JavaScript form validation
Connect to database (MySQL / Firebase)
Show confirmation message after submission
Add appointment scheduling system
